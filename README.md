# 北京理工大学模电实验报告模板（latex）
因为觉得老师给的word版文档排版起来很痛苦，我和ai合作，基于word模板和格式要求创建了一个可以满足老师要求的latex模板！这样大家就可以免去排版的烦恼了！
当然，基于本人水平有限，它肯定不是一个很成熟的模板。如果大家有更好的模板或者改善方式，欢迎提出批评指正！


下载方式：（首先确保已经下载好latex）

1.点击右上角code选项，找到最下面download ZIP 并下载；

2.解压缩后找到“模电实验报告模板.tex”文件并打开，就能看到模板形式了！

3.根据个人情况自行修改报告，里面已经有示例内容。

# 注意事项：

使用xelatex方式进行编译，一般都是pdflatex，这个记住要改一下，不然会报错，很重要!

方式自行查询，很简单。
